# Harsh Panchal Portfolio

This is a Vercel-ready Next.js portfolio site for Harsh Panchal. It is designed to support an entry-level IT support and technical support job search while also showcasing software development projects.

## Highlights

- Custom one-page portfolio design
- Contact section for recruiters and hiring managers
- Featured projects including the ResolveIT help desk system
- Built with Next.js so it can deploy easily on Vercel with a custom domain

## Run Locally

```powershell
cd C:\Users\harsh\OneDrive\Desktop\code\harsh-portfolio
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build Check

```powershell
npm run build
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Import the repository into Vercel.
3. Set the production domain in Vercel project settings.
4. Point your custom domain DNS to Vercel.
